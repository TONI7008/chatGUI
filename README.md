
# Chat GUI

A modern C++ chat application with graphical user interface.

## Features

- Real-time messaging
- User-friendly interface
- Cross-platform support
- Lightweight and efficient

## Installation

1. Clone the repository
2. Install dependencies (Qt or your preferred GUI framework)
3. Build the project:
    ```bash
    mkdir build
    cd build
    cmake ..
    make
    ```

## Usage

```bash
./chatGui
```
